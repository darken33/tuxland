INTRODUCTION
------------

Mr Bill, ayant peur de la concurrence a dérobé l'ensemble des CD originaux 
des distributions Linux. Tux entreprend donc de le traquer à travers les 5 
continents afin de les récupérer.

------------------------------------------------------------------------------

LES BASES DU JEU
----------------

Le jeu fonctionne à la fois sous GNU/Linux et sous Windows, il necessite q'un
JRE Java soit installé sur votre machine, vous pouvez en trouver un sur

Sous Windows : Il vous faut executer le script tuxland.bat

Sous Linux : Il vous faut executer le script tuxland.sh 

------------------------------------------------------------------------------

LES BASES DU JEU
----------------

Tux doit se déplacer dans un niveau qui est chronométré à l'aide des 3 flèches
Up (sauter), Left (se deplacer à gauche) et Right (se déplacer à Droit) afin 
de récupérer l'ensemble des CD dspersés dans celui ci. Il doit toutefois 
eviter les contacts avec les pièges ou ennemis. Dans le cas ou il touche un 
piège ou un ennemis, Tux perd une vie (3 aux depart). Enfin si tux ne 
parvient pas à recupérer les CD avant que le chronomètre tombe à zéro, Tux 
perd aussi une vie.

------------------------------------------------------------------------------

LES COMMANDES DU JEU
--------------------

Souris        => Navigation dans les menus
Fleche gauche => Aller vers la gauche
Fleche droite => Aller vers la droite
Fleche haut   => Sauter
Fleche bas    => Faire l'autruche (permet d'éviter un ennemi)
Touche shift  => Courir
Touche P      => Mettre/Enlever la Pause
